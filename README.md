##  **Project Overview**

**Cinematch** is a modern, responsive movie recommendation web application built with **React.js**. It delivers personalized movie suggestions based on user preferences and moods using real-time data from public movie REST APIs.

**Key Skills Demonstrated:**
- React.js component architecture
- REST API integration with Axios
- Responsive UI design
- Modern JavaScript (ES6+)
- Performance optimization

##  **Features**

 **Personalized Recommendations** based on user mood
 **Real-time API data** fetching with Axios
 **Fully Responsive** - Desktop, tablet, mobile
 **Dynamic movie search** and filtering
 **Modular React components**
 **Modern React Hooks** & Context API

## **Tech Stack**
**Core Technologies:**
- **React.js** - Component-based UI architecture
- **Axios** - HTTP client for API requests
- **HTML5/CSS3** - Semantic markup & modern styling
- **JavaScript ES6+** - Modern language features
